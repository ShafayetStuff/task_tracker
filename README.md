<!-- GETTING STARTED -->
## Getting Started

Easy and peasy. You should have Python installed.

### Installation

1. Clone the repo
   ```sh
   git clone https://github.com/ShafayetStuff/task_tracker
   ```
2. cd into the directory and run
   ```sh
   py .\task_tracker.py
   ```
